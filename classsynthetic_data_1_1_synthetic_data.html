Detailed Description

Class: SyntheticData.

Creates a synthetic dataset, i.e. cutouts of original plans, overlaid with synthetic (fake) symbols.

Creates a synthetic plan from a cutout image and a list of symbols.

To avoid a large variance in the aspect ratio of symbols, all synthetic symbols are scaled to the average width and height of all original symbols in a cutout.

Parameters

Returns
<dict<image object, list<list<double>>, list<str>, str, int, int>> - A dictionary.
Note
The output dictionary is containing the following:
  • The synthetic plan image
  • A matrix containing the bounding boxes of all symbols
  • The corresponding symbol names (labels)
  • Original plan's name
  • xMin of cutout in original plan
  • yMin of cutout in original plan

◆ __getSynthSymbols()

def syntheticData.SyntheticData.__getSynthSymbols (   self,
  numOriginalSymbols 
)
private

Finds symbols at random from the class property symbolsToIncrease.

To avoid symbols overlaying one another, the number of symbols per cutout is soft-limited to 10, i.e. total number of symbols is limited to 10 if the number of original symbols is lower than 10, else total number of symbols = number of original symbols + synthSymbolsPerCutout. All symbols image objects are imported using the function __importSymbolFiles()

Note
If no symbols found in the property symbolsToIncrease, the property is reset.
Warning
Function returns 0 symbols if the synthetic data is created out of a validation dataset.
Parameters
numOriginalSymbols:intNumber of original symbols in cutout.
Returns
dict<list<str> , list<image object>> Symbol image objects and labels.
