def segmentationModel.SegmentationModel.__compileUNet (   self)
private

This method compiles the U-Net model using the specified optimizer and loss function.

The mean intersection over union (IoU) is used as a metric for evaluating the model performance.

The summary of the compiled model is also printed after compilation.

Returns
No return.
Note
The optimizer and loss function used to compile the model are specified as instance variables in the class.
This method is called in the constructor to compile the U-Net model immediately after its creation.

This method trains the model on the given training data and saves the best model to disk.

Parameters
trainPath:strPath to the training data.
testPath:strPath to the test data.
IMG_HEIGHT:floatHeight of the images in the data.
IMG_WIDTH:flaotWidth of the images in the data.
IMG_CHANNELS:intNumber of channels in the images.
Note
The training process is done using the Adam optimizer and binary crossentropy loss.
The training progress is monitored using the Intersection Over Union (IoU) metric.
Warning
The training process may take a long time and may require a large amount of memory, especially for large images.
Exceptions
ExceptionIf the shape of the input image is not the same as the shape of the images the model was trained on.
