Detailed Description

: main.py Main File.

Run file with the following arguments to use program.

  • python main.py train -> Start training a new neural network.
  • python main.py train continue -> Continue training from a saved checkpoint (bestModel.pth must exist in the outputs directory - outputsDir in config.py).
  • python main.py test -> Evaluate a testing dataset, export results as an XML file, and overlay results onto testing images.
  • python main.py label -> Evaluate a testing dataset and export results as an XML file. Contains all settings for ML model training.
