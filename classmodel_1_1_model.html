Detailed Description

Class: Model.

Container for the neural network (NN). Run trainModel() to start training the model.

Warning
To avoid runtime errors make sure the labelled classes can be found in the defined classes list (variable classes in config.py).

Initializes a Model object.

Object can be initialized with either an SSD or an FRCNN neural network (NN).

To initialize class with a partially trained neural network, make sure the argument trained is set to True.

Remaining configuration variables can be found in config.py.

Parameters
numClasses:intNumber of classes. Cannot be zero.
trained:boolChecks whether a pre-trained NN (pre-trained with the COCO dataset) should be used as the initial NN (epoch 0). Variable is redundant if continueTraining is set to True.
See also
https://cocodataset.org/
Parameters
Kwargs
continueTraining:boolIndicates whether training is continued from an existing NN (true) or starting with an untrained NN* (false). If set to True, make sure lastModel.pth exists in the output directory. Path to the output directory can be changed in config.py.
Returns
Model object.

def model.Model.trainModel (   self)

Main method to start training the neural network.

Both training and validation datasets are created and evaluated here (two private methods).

Optimizer is used to adjust the model parameters from training dataset.

Model generated after every epoch is saved as lastModel.pth in the output directory.

Model with the lowest validation loss is stored as bestModel.pth in the output directory.

Losses plot is updated every epoch by calling HelperFunctions.saveLossPlot().

Returns
No return.
