Detailed Description

Class: AccuracyChecker.

Deprecated:
USE AccuracyChecker in accuracyUpdated.py

Used to check the accuracy of the model's prediction against the ground truth (labelled benchmark). Benchmarks must be Pascal VOC XML files.

Use the label option when initializing class: Test before initializing this class.

Note
All bounding boxes are ordered as follows: xMin, yMin, xMax, yMax.

Initializes class: AccuracyChecker.

Path to the benchmark results can be edited in config.py. Class works as a callable object, i.e. class implicitly generates two bar plots when initialized:

  • Average Intersection over Union for each class
  • Occurrence frequency of each class

Both Plots will be found in the outputs directory (outputDir in config.py).
