Detailed Description

Class: CutoutCreator.

Creates cutouts from original plans.

Acts as a callable object with state.

Note
The "state" is the cutouts.

Initializes CutoutCreator object.

Cutouts are created directly within the constructor. Cutouts are done in a two-step process.

Parameters
planPath:strPath to directory with original plans.
cutoutsPerAxis:intNumber of cutouts per axis (x or y).
iOAThreshold:doubleIntersection over Area threshold of symbols on the boundary of cutout. Bounding boxes and labels with IoA < iOAThreshold are ignored.
Returns
CutoutCreator object, including the list of cutouts as one of the object's properties.
Example
cutoutsPerAxis = 4
  1. Step: 16 cutouts are created
  2. Step: 9 cutouts are created, shifted by width/4/2 and height/4/2 in x and y, respectively

Crops an image using minimum and maximum values in both x and y directions.

Calls __findsSymbolsInCutouts() to extract the symbol information (bounding boxes, labels, width and height).

Parameters
plan:imageobjectOriginal plan.
xMin:intMinimum value in x.
yMin:intMinimum value in y.
xMax:intMaximum value in x.
yMax:intMinimum value in y.
iterator:intIndex of plan in the class property plans.
Returns
<dict<image object, list<list<double>>, list<str>, double, int, int, int, str, int, int>> - A dictionary containing the following (not in order): Cutout of plan; the bounding box, label, with, and height of the all symbols in cutout; the cutout's blur variance; the cutout's xMin and yMin in original plan; and the original image's name.
Note
The dictionary contains the following (not in order):
  • Cutout of plan
  • Bounding box, label, width, and height of the all symbols in cutout
  • The cutout's blur variance
  • The cutout's xMin and yMin in original plan
  • The original image's name.

For all plans found in the property planFolder, initializes an image object, extracts the plan's name and calls __getSymbolLocations().

The results are then stored in three separate lists. All lists are stored as class properties.

Returns
No return.
Exceptions
ImportError
Warning
The order of the lists matters. Do not use a sorting function on them.
