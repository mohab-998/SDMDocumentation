Detailed Description

Class: PlanDataset.

Dataset class used to generate datasets for both training and validation. Do not use the constructor to initialize this class, use the static method createTrainingDataset() and createValidationDataset() to initialize this class. All arguments can be edited directly from config.py.

Note
Class acts as a container for a list of training/validation data, i.e. individual plans can be accessed using the [] operator.
